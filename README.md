#  Tony's AI Data & Vision Business Dashboard

An advanced AI-powered dashboard that integrates Multi-modal LLMs, Computer Vision, and Sentiment Analysis to automate business workflows.

##  Key Features
* **Vision Engine (LangChain + Ollama):** Analyze screenshots and images locally using Llama 3.2 Vision.
* **Sentiment Analysis (Hugging Face):** Real-time emotion detection for user queries using specialized Transformer models.
* **Data Analytics (Pandas):** Instant statistical summaries and AI analysis of CSV/Excel business data.
* **Computer Vision (OpenCV):** Image pre-processing (Grayscale & Resizing) for faster AI inference.
* **Workflow Automation (Zapier):** Seamlessly send AI-generated insights to email/CRM via Webhooks.

##  Tech Stack
- **Framework:** Streamlit
- **Orchestration:** LangChain
- **AI Models:** Ollama (Llama 3.2 Vision), Hugging Face (DistilBERT)
- **Data/Vision:** Pandas, OpenCV, NumPy
- **Automation:** Zapier Webhooks
